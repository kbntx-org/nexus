# Portfolio Frontend

A modern, responsive portfolio website built with Angular 20, showcasing professional experience, projects, and skills. The application features a clean design with dark mode support, smooth animations, and optimized performance.

## Overview

The Portfolio Frontend is a single-page application (SPA) that serves as a personal portfolio website. It demonstrates modern web development practices, platform engineering principles, and showcases technical expertise through a well-structured, maintainable codebase.

## Key Features

- 🎨 **Modern UI/UX**: Clean, responsive design with dark mode support
- ⚡ **Performance Optimized**: Fast loading times with optimized assets and caching strategies
- 📱 **Fully Responsive**: Works seamlessly across all device sizes
- 🌙 **Dark Mode**: System preference detection with manual toggle
- 🎭 **Smooth Animations**: Engaging user experience with CSS animations
- 🚀 **Production Ready**: Containerized with Caddy for efficient deployment
- 📊 **Structured Content**: Organized sections for home, experiences, projects, and CV

## Sections

- **Home**: Landing page with skills overview and introduction
- **Experiences**: Timeline of professional work experience
- **Projects**: Showcase of notable projects with details
- **CV**: Resume display with downloadable PDF option
