# First week checklist

- [ ] Clone the repo and install dependencies.
- [ ] Run format and lint checks.
- [ ] Run portfolio tests.
- [ ] Read platform overview diagrams.
- [ ] Inspect ArgoCD app-of-apps values for deployed components.
- [ ] Make one small PR touching app code and one touching platform config.

## Done criteria

You are onboarded when you can explain:

- how code moves from PR to deployment,
- how to find the owning folder for a running component,
- how to verify a change safely before merge.
